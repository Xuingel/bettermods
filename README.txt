--How to make a server on your PC:--
Open this link and donwload latest FORGE 1.18.1 installer
-> https://files.minecraftforge.net/net/minecraftforge/forge/ 

--Open the forge installer and install server into a new folder 
> Drag Mods Folder, Config Folder and server.properties into the new folder you made 
> Open the  the run.bat
> After it says click any button to continue, do that

> Launch the run.bat file again


Q: I'm trying to run a server but it's not working, how do I fix this?
A: You need to use Java 17, 64-bit to run a modded Minecraft server. You can download it from here: https://adoptopenjdk.net/ - select OpenJDK 17 (Latest)
